# Dasboard-Dlesep-Coffe

# ☕ Dlesep Coffee Shop Sales Dashboard (USA Branch)

## Project Overview
This repository contains my first-ever data visualization project: an interactive Sales Dashboard for **Dlesep Coffee Shop**. This project focuses on analyzing transaction patterns, customer behavior, and product performance to drive business growth.

The dashboard was built using **Microsoft Excel**, focusing on data cleaning, pivot tables, and UI/UX design for reporting.

## 📊 Key Business Insights
Based on the analysis of the dataset (January - June):
- **Peak Hours:** High customer activity starts at 08:00 AM, peaking between **10:00 AM – 11:00 AM**.
- **Busy Days:** Weekends are the peak periods, with **Sunday** being the highest sales day.
- **Customer Preferences:** **45%** of customers prefer the **Large** size.
- **Top Performer:** **Sanka Coffee** is the primary revenue driver among all products.
- **Global Reach:** Performance remains consistent across all store locations in the USA branch.

## 🛠️ Tools & Skills
- **Excel:** Pivot Tables, Power Query (for data cleaning), Slicers, and Advanced Charting.
- **AI Integration:** Used AI to generate data logic flowcharts for better documentation.
- **Data Analytics:** Trend analysis, distribution analysis, and business metric calculation (Avg Bill/Person, Avg Order/Person).

## 📂 Repository Contents
- `Dlesep_Coffee_Dashboard.xlsx`: The main interactive Excel file.
- `Dataset_Coffee_Shop.csv`: The raw transaction data used for analysis.
- `Flowchart_Logic.png`: A visual representation of the data processing flow.

## ⚠️ Known Issues

## 🚀 How to Use
1. Clone this repository.
2. Open the `.xlsx` file in Microsoft Excel.
3. Enable Macros/Content if prompted.
4. Use the **Month Name Slicer** on the left to filter the data.
